# NALLET — Sistem Informasi Keuangan Pribadi

> Platform pengelolaan keuangan pribadi berbasis web yang membantu pengguna memantau saldo, transaksi, anggaran, dan tujuan keuangan dengan tampilan yang modern dan responsif.

---

## Fitur

- **Dashboard** — Ringkasan saldo, pemasukan, pengeluaran, dan grafik aliran uang bulanan
- **Transaksi** — Daftar lengkap transaksi dengan filter, kategori, dan status
- **Dompet** — Manajemen akun bank (BNI, BCA, Mandiri) beserta statistik pengeluaran
- **Tujuan** — Tracking tabungan untuk tujuan finansial (gadget, kendaraan, liburan, dll)
- **Anggaran** — Pemantauan anggaran per kategori dengan visual circular progress
